## 🚀 How to Test

1. **Start XAMPP**: Open XAMPP Control Panel and start Apache & MySQL
2. **Open Browser**: Go to `http://localhost/CampusHub/shopswift/`
3. **Test Navigation**: Click through all navigation links
4. **Test Dropdowns**: Hover over Women, Men, Accessories
5. **Test Interactions**:
   - Click "Add to Cart" on any product
   - Click heart icon to add to wishlist
   - Try dark mode toggle
